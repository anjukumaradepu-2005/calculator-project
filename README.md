# Calculator Project

A clean and simple calculator built using HTML, CSS, and JavaScript.

## 🚀 Features
- Basic operations (+, -, *, /, %)
- Keyboard support
- Delete and clear functions
- Responsive UI

## 🛠️ Technologies

- CSS
- JavaScript
